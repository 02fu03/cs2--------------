# CS2 单饰品价格演化及趋势预测研究

项目目录包含用于抓取、处理、建模与预测 CS2（Counter-Strike 2）饰品历史价格数据的脚本、数据与训练好的模型文件。

## 项目简介

本仓库旨在研究单个饰品的价格演化规律并基于历史数据进行短期（如7天）价格预测。代码包含数据采集、特征处理、模型训练/保存、预测与评估相关脚本，以及若干示例数据与训练好的模型文件，便于复现与进一步实验。

## 主要文件说明

- 数据 / 缓存
  - `AK47抽象派1337(久经沙场).csv`、`M4A1消音版  梦魇 (崭新出厂)_悠悠有品_近3个月数据.csv`：样例原始数据 CSV。
  - `cache_*.csv`：抓取或预处理的缓存数据。
  - `CS2_M4A1消音版_20260608_012344.txt`：原始抓取或导出文本数据示例。

- 脚本
  - `获取数据集.py`：用于批量获取或合并数据集的脚本。
  - `获取good id.py`、`查找good id.py`：用于查找与映射饰品 ID（good id）的工具脚本。
  - `模型预测7天.py`：使用训练好的模型进行 7 天价格预测的主脚本（生成 `pred_*.html` 或 CSV 输出）。
  - `模型评估.py`：对预测结果与交叉验证结果进行评估并生成 `cv_results_*.csv`。

- 模型与预测结果
  - `model_*.joblib`：训练并保存好的模型权重/对象（例如 `model_AK47抽象派1337_..._h7_tuned.joblib`）。
  - `forecast_*.csv` / `forecast_*_tuned.csv`：预测输出 CSV。
  - `pred_*.html` / `forecast_*_compare_chart.html`：用于展示预测与真实值对比的 HTML 可视化结果。

- 其他
  - `AK-47 ID.txt`：饰品 ID 列表或映射参考文件。
  - `requirements.txt`：Python 依赖列表。

## 环境与依赖

建议创建虚拟环境并安装依赖：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install --upgrade pip
pip install -r requirements.txt
```

> 如果你的系统为 cmd，请将激活命令替换为 `\.venv\Scripts\activate.bat`。

## 快速使用说明

1. 准备数据：将原始 CSV 放入项目目录，或运行 `获取数据集.py` 来抓取/合并数据。

```powershell
python 获取数据集.py
```

2. 预测（使用现有模型）：运行 `模型预测7天.py`，脚本默认会加载同目录下的 `model_*.joblib` 并输出 `pred_*.html` 或 `forecast_*.csv`。

```powershell
python 模型预测7天.py
```

3. 评估：运行 `模型评估.py` 来生成评估结果 CSV（`cv_results_*.csv`）并查看模型性能对比。

```powershell
python 模型评估.py
```

## 输出说明

- `pred_*.html`：每次预测的交互式或静态可视化，便于在浏览器中查看预测与历史真实价格对比。
- `forecast_*.csv`：包含预测的时间序列数值结果，可用于后续分析或上链保存。
- `cv_results_*.csv`：交叉验证与超参搜索的结果表。

## 复现实验与训练（可选）

如果你希望从头训练模型：

1. 准备干净的训练数据（参见脚本 `获取数据集.py`）。
2. 修改或补充训练脚本（当前仓库可能将训练逻辑内嵌在处理或评估脚本中）。
3. 运行训练脚本并保存为 `model_*.joblib`。
